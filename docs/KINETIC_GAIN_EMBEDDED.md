# Kinetic Gain Embedded Tie-back

This repo demonstrates the Tableau permission-audit control-plane shape of Kinetic Gain Embedded:

- operator-safe visibility into workbook permission overrides, broken group sync, stale certification proof, expired external shares, and audit continuity
- one dashboard joining access trust, review ownership, and remediation packets
- a clear path from public proof repo -> paid template -> hosted preview -> embedded product engagement

For embedded product work, the same primitive becomes an in-app analytics-administration panel where operators need evidence-rich Tableau governance without direct access to raw admin consoles or live tenant credentials.
