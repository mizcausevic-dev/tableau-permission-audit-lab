# Security Policy

## Data handling

- This repo only ships synthetic Tableau permission, certification, sharing, and audit samples.
- Do not commit live user exports, workbook URLs, group membership dumps, activity logs, or embedded secrets.
- Tableau OAuth tokens, personal access tokens, session cookies, and site admin credentials must stay outside the repo.

## Claims discipline

- This repo demonstrates governance posture and audit-readiness patterns.
- It does **not** claim compliance certification, production tenant coverage, or live customer enforcement by itself.
